# CachedWarps

This plugin allows you to manage your warps in cache. Only access to file when creating or deleting a warp.

## Commands & Permissions
```text
/setwarp <name> - cachedwarps.set
/delwarp <name> - cachedwarps.delete
/warp <name>
/warps or /listwarps
/acw [reload] - cachedwarps.admin
```
